--------------------------------------------------

Score the project from 0 → 100 using:

Examples of violations to detect:

- Missing validation (commands without validators)
- Business logic inside controllers
- Direct Infrastructure usage in API/Application

- Sync-over-async (.Result / .GetAwaiter().GetResult())

- Dead code / unused usings
- Namespace inconsistencies
- Missing cancellation tokens
- Missing interfaces for services
- Anemic domain model
